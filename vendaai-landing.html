FAQ
Perguntas frequentes
Preciso de saber programação para usar o VendaAI? +
Não! O VendaAI foi desenhado para qualquer pessoa. Basta criar a conta, adicionar os teus produtos e ligar o WhatsApp. Não é necessário nenhum conhecimento técnico.
O bot consegue negociar preços automaticamente? +
Sim! Podes configurar margens de negociação para cada produto. O bot usa inteligência artificial para negociar dentro dos limites que defines, sempre protegendo a tua margem de lucro.
Quais formas de pagamento são aceites? +
O VendaAI integra com Multicaixa Express e Unitel Money. Estamos a trabalhar para adicionar mais métodos de pagamento angolanos em breve.
Os dados dos meus clientes estão seguros? +
Sim. Todos os dados são encriptados e armazenados em servidores seguros. Nunca partilhamos dados de clientes com terceiros. A tua privacidade e a dos teus clientes é a nossa prioridade.
Posso cancelar a qualquer momento? +
Sim, podes cancelar a tua subscrição a qualquer momento sem penalizações. O teu plano fica activo até ao fim do período já pago.
Funciona para qualquer tipo de negócio? +
Sim! O VendaAI funciona para qualquer tipo de negócio: electrónica, moda, alimentação, imobiliária, serviços e muito mais. Se vendes pelo WhatsApp, o VendaAI é para ti.
